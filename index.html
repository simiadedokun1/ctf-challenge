Beginner: Find the Flag in the Page Source

Difficulty: Beginner — suitable for first-time CTF players. Open the page source (View > Page Source or Ctrl+U) and hunt for three pieces.

Hints (don't peek if you want to solve it yourself):

  • There are three parts, each encoded differently.
  • Common encodings include Base64, hex, and ROT13.
  • Combine the decoded parts with underscores: CTF{part1_part2_part3}.

If you want to test your decoding skills locally: try base64 --decode, xxd -r -p (or python), and tr 'A-Za-z' 'N-ZA-Mn-za-m' for ROT13.

Good luck — and remember: look everywhere in the source (comments, attributes, CSS, JS).
